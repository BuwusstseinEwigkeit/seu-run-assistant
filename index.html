    本地工具使用说明
    先准备 Token 和照片，再按需要选择单次或批量提交

    额外检查、课表导入、失败重试都是可选功能；不启用时不会增加主流程负担。

    查看 Github 最新教程 Token 形如 Bearer [part1].[part2].[part3]
    快速流程
    1. 获取并填写 Token

      Token 用于校验当前账号。输入框会先做格式检查，提交前还会请求接口确认是否有效。

    2. 选择场地和时间

      单次模式可以手动选择时间，也可以点随机时间。时间会按场地规则和配速范围做基础校验。

    3. 上传开始和结束照片

      照片会在页面内预览。提交前一致性检查可读取 EXIF 时间，并给出白天/夜间匹配提示。

    4. 按需启用额外检查

      课表冲突、照片时间阻止提交都默认关闭。需要时先导入课表，再勾选对应开关。

    5. 提交或批量提交

      批量模式支持照片池、跳过周末、随机场地、失败记录保存和重新提交失败日期。

    课表与一致性检查
    手动添加

    选择星期、开始时间、结束时间后添加。导入后的每条课表也可以直接修改。

    CSV 导入

    支持 weekday,start,end,name 或 星期,节次范围,课程名称。

    粘贴文本

    可以粘贴网页表格、CSV 文本，或每行类似“星期一 1-2节 高等数学分析II”。

    可选阻止

    只有勾选“提交时启用这些额外检查”和对应阻止项后，冲突才会阻止提交。
