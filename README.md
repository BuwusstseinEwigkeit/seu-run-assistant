# ML SEU Exercise Helper

## 概述
本项目旨在帮助 SEU 的本科生更好地进行课外锻炼，完全开源，请勿商业化，请自觉遵循 [GPLv3 许可证](LICENSE)。

本项目有着简洁美观的 WebUI, 操作简单，是本科生们课外锻炼的得力助手。

如果大家喜欢本项目，请给颗小星星哦~

欢迎 Star、Fork 和 PR！

## 功能范围

本版本基于 [原版 (2025-11-04 stable)](past-versions/ml-seu_exercise_helper-20251104-stable.html) 调整，目标是提供一套功能完整、可直接在浏览器中使用的课外锻炼记录辅助工具。

### 核心功能要求

| 模块 | 功能要求 | 验收标准 |
|------|----------|----------|
| 单次提交 | 支持 Token 输入、场地选择、起止时间设置、开始/结束照片上传和记录提交 | 用户按页面步骤填写后，可以完成一次跑步记录提交 |
| 随机轨迹 | 支持按所选场地生成轨迹点 | 生成结果包含轨迹点、距离、用时等信息，并能用于提交 payload |
| 随机时间 | 支持根据当前轨迹自动生成开始和结束时间 | 生成时间满足场地规则、运动时长规则，并与轨迹用时联动 |
| 批量提交 | 支持按日期范围连续提交，可选择跳过周末 | 批量过程中显示进度，结束后展示成功和失败数量 |
| 随机场地 | 批量模式可在已有场地中随机选择 | 每天提交前能重新生成对应场地轨迹 |
| 照片池 | 批量模式支持开始照片池和结束照片池 | 每次提交前从照片池中随机选取图片 |
| 本地校验 | 提交前检查 Token 格式、时间范围、照片是否已选择 | 常见参数缺失时给出页面提示，不直接发起提交 |
| 结果反馈 | 上传成功、失败、批量完成时给出弹窗提示 | 用户能看到当前操作是否完成，以及失败原因摘要 |

### 功能边界

- 项目优先保证浏览器页面上的主要流程完整：单次提交、随机轨迹、随机时间、批量提交。
- 随机轨迹基于内置场地边界生成，目标是让轨迹落在场地范围附近并满足提交数据结构要求。
- 时间、距离、配速会做基本联动，但不承诺模拟真实运动设备的全部细节。
- Token 只在当前页面会话中使用，不提供账号登录、Token 管理或云端保存能力。
- 本项目不内置后端服务，所有提交动作均由浏览器直接请求目标接口。

### 已知限制

- 浏览器可能会限制部分请求头，实际请求结果以目标接口和当前浏览器策略为准。
- 批量提交依赖网络稳定性，中途失败时需要根据失败日期重新操作。
- 随机生成的数据用于辅助填写记录，不替代用户对记录真实性和合法性的判断。

## 使用方法
下载 [Release](https://github.com/midairlogn/ML-SEU-Exercise-Helper/releases) 中的最新版本，用浏览器打开 `.html` 。(推荐在电脑上操作)

### 单次提交
1. 填入你的 Token
2. 选择场地，点击 `随机轨迹` 生成符合要求的轨迹
3. 点击 `随机时间` 或手动选择时间
4. 上传两张照片作为开始、结束照片
5. 点击 `提交跑步记录`

### 批量提交
1. 填入 Token
2. 切换到"批量上传"标签页
3. 选择日期范围（可跳过周末）
4. 上传照片池（至少各一张）
5. 点击 `开始批量提交`

> 照片采用自拍，人脸尽量占据整个画面。可以使用相同的照片，但是一定要本人的。

> **需要校园网**，**在成功提示窗口弹出前切勿离开、关闭页面**！

### Token获取说明
> Token形如 `Bearer [part1].[part2].[part3]`。

1. 打开 [https://tyxsjpt.seu.edu.cn/h5/#/pages/home/index](https://tyxsjpt.seu.edu.cn/h5/#/pages/home/index) ，登录账号。

2. 按 `F12` (有的电脑是 `Fn + F12` )打开浏览器的开发人员工具，进入 `控制台` ( `console` )。

3. 展开 `request res========` 后面的 `object` ，找到 `request.headers.token` 。

4. 右键复数字符串内容。**注意: 不需要引号!**
